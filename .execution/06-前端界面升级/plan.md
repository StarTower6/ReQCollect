# Plan v1: 06 — 前端界面升级

## 1. 任务理解
- **需求来源**: `docs/requirements/06-前端界面/README.md`
- **核心目标**: 在保留现有 CSS 设计语言的前提下，升级前端为三栏布局，新增需求画像面板
- **关键设计决策待确认**: 见下方第 8 节

## 2. 改动范围（最终确认）

### P0 — 布局+画像（必须做）
| 改动 | 说明 |
|------|------|
| 三栏布局 | 左侧会话列表 + 中间对话区 + 右侧画像面板 |
| 技术路线 | Vue 3 CDN + Element Plus CDN，零构建 |
| 设计约束 | 完全保留现有 CSS 设计变量和配色体系 |
| 需求画像面板 | 完整度进度环 + 11 字段状态列表 |
| 字段交互 | 点击字段展开详情、缺失字段引导 |
| 响应式 | 大屏固定三栏，小屏右侧折叠成抽屉按钮 |

### P1 — 对话增强
| 改动 | 说明 |
|------|------|
| 消息头像 | 用户/AI 分别显示头像 |
| 消息时间戳 | 每条消息显示时间 |
| 代码高亮 | highlight.js 集成 |
| 消息复制 | 悬停显示复制按钮 |
| 流式打字效果 | 逐词出现动画 |

### P1 — PRD 预览页
| 改动 | 说明 |
|------|------|
| PRD 独立预览页 | 左侧目录导航 + 章节定位 |
| 下载按钮 | Markdown / Word / PDF |

### 明确不做
- ❌ 暗色模式（P2）
- ❌ 仪表盘/搜索/日期分组（P2）
- ❌ 引入 Vite/Webpack 等构建工具（仅 CDN）

## 3. 技术方案

### Vue 3 CDN + Element Plus CDN（已确认）
- Vue 3 Global Build: `unpkg.com/vue@3/dist/vue.global.prod.js`
- Element Plus: CDN UMD + CSS
- Markdown 渲染: marked.js
- 代码高亮: highlight.js
- 零构建，零 npm，单 HTML 文件
- **所有 CSS 变量 100% 保留**，Element Plus 仅用于组件结构，样式覆盖为现有设计系统

## 4. 文件结构（Vue 3 版本）

考虑到保持单文件部署，使用 Vue 3 CDN 的 **options API** 组织组件：

```html
static/index.html
├── <style>     ← 现有全部 CSS 变量 + 新增布局样式
├── <div id="app">  ← Vue 3 根组件
│   ├── <layout>
│   │   ├── <session-list>   左右: 会话列表
│   │   ├── <chat-area>      中间: 对话区
│   │   └── <profile-panel>  右侧: 画像面板（可折叠抽屉）
│   └── <prd-preview>        PRD 预览页（独立视图）
└── <script>
    ├── Vue 3 createApp
    ├── Element Plus 注册
    └── 组件定义 + API 调用
```

## 5. 验收标准（最终版）

- [ ] 三栏布局可正常显示：左侧对话列表 ≤260px，中间对话区自适应，右侧画像面板 ≤320px

- [ ] 切换会话时右侧画像内容同步更新
- [ ] 所有现有对话功能不受影响（发送/流式/快捷回复）

## 5. 风险
- 右侧面板宽度增加可能挤压对话区空间
- 现有 CSS 选择器可能与新增布局冲突，需隔离
- 响应式方案在 <768px 设备上的体验需权衡
