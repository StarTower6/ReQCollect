交互说明

1. 展开 workspace — 该 workspace 关联了多少文件就显示计数。展开后先展示文件目录树，虚线分隔后展示对话记录
2. 文件图标 — 📝 md / 📘 docx / 📊 xlsx / 📄 txt / ⚙️ yaml / 📋 json
3. 文件行「+」按钮 — 点击将文件引用注入当前对话，输入框上方出现蓝色引用标签
4. 点击文件名 — 右侧弹出预览窗口（Markdown 渲染 / Office 文本预览），非模态，不打断对话
5. AI 生成标记 — _generated/ 目录下的 AI 产出文件带绿色「AI」标签
6. @ 引用 — 输入框输入 @ 弹出搜索面板，搜索并选择工作区文件引用
7. 右键菜单 — 右键文件弹出：预览、引用、分析、删除
