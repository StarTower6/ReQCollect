交互说明

1. 左侧不变 — 工作空间 + 会话树，保持原有交互
2. 右侧文件树 — 展示当前选中 workspace 的文件，按目录结构展开。切换左侧 workspace 时右侧自动切换
3. ⊕ 引用按钮 — 文件行 hover 后出现的 + 按钮，点击将文件引用注入对话，右侧底部「当前引用」区域显示
4. 无 workspace 选定 — 右侧显示空状态：「请选择一个工作空间」
5. 右键菜单 — 预览 / 引用 / 分析 / 删除
6. @ 引用兼容 — 输入框 @ 弹出搜索面板与右侧文件树互补
7. 上传入口 — 右侧顶部「+ 上传」按钮（替代原来 WorkspaceDetail Tab 中的上传按钮）
8. 右侧替换需求画像面板 — 当前右侧 ProfilePanel 位置被文件树替代。需求画像面板改为通过头部按钮或快捷键切换
