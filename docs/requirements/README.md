# ReQCollect — 需求文档索引

> 项目：企业级业务需求采集与分析平台
> 版本：v0.1.0 → v1.0.0（企业级）
> 数据库：MySQL 8.0
> 基于：ReQCollect（Apache 2.0）

---

## 功能架构总览

```
用户接入层
  ├── Web UI（浏览器）
  └── 企业微信（WeCom Bot）

核心引擎层
  ├── 对话挖需引擎（Phase 1）
  │   ├── ReAct Agent → 多轮对话引导
  │   ├── 完整度评估 → 11字段加权评分
  │   └── 规则提取器 → 正则保底识别
  └── PRD 生成管线（Phase 2）
      ├── Planner → 章节规划
      ├── Generator → LLM 逐章生成
      └── Assembler → Markdown 拼装

基础设施层
  ├── 用户与权限 → LDAP/OIDC + RBAC + 审计
  ├── 存储与数据 → MySQL 8.0 + 备份 + 搜索
  └── 部署与运维 → Docker + Nginx + 监控

集成层
  ├── 企业微信 → 对话 + 通知 + 扫码登录
  └── HtmlHub → PRD 自动发布
```
