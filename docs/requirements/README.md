# ReQCollect — 需求文档索引

> 项目：企业级业务需求采集与分析平台
> 版本：v2.0.0（已实现基础管线 + 用户认证 + MySQL 存储 + Docker 部署）
> 数据库：MySQL 8.0
> 基于：ReQCollect（Apache 2.0）

---

## 功能架构总览

```
核心引擎层
  ├── 对话挖需引擎（Phase 1）
  │   ├── ReAct Agent → 多轮对话引导
  │   ├── 完整度评估 → 11字段加权评分
  │   └── 规则提取器 → 正则保底识别
  └── PRD 生成管线（Phase 2）
      ├── Planner → 章节规划
      ├── Generator → LLM 逐章生成
      └── Assembler → Markdown 拼装

基础设施层 ✅ 已实现
  ├── 用户与权限 → JWT + RBAC + 用户管理
  ├── 存储与数据 → MySQL 8.0 + JSON fallback
  └── 部署与运维 → Docker Compose + Nginx

新增功能层
  └── 导入与文档库 → 会议记录导入分析 + 历史 PRD 上传
```
