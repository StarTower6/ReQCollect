# 06 — 前端界面升级

## 概述

当前为单页原生 HTML/CSS/JS 前端。升级为 Vue 3 CDN + Element Plus CDN 组件化架构，
保留现有全部 CSS 设计变量和配色体系，零构建工具。

## 技术约束
- Vue 3 CDN：`unpkg.com/vue@3/dist/vue.global.prod.js`
- Element Plus CDN：UMD + CSS
- Markdown：`marked.js`
- 代码高亮：`highlight.js`
- ❌ 不引入 Vite/Webpack 等构建工具
- ❌ 不替换现有 CSS 设计系统
